# Tapt Payment Gateway for WooCommerce

Transform your WooCommerce store with modern payment processing featuring NFC tap-to-pay, QR code payments, and transparent fixed-fee pricing.

## ✨ Features

- **🔥 NFC Tap-to-Pay**: Customers tap their phones to pay instantly
- **📱 QR Code Payments**: Quick scanning for contactless transactions
- **💰 Fixed Fee Structure**: Just $0.25 per transaction, no percentages
- **🔒 Bank-Level Security**: PCI compliant with encrypted data transmission
- **⚡ Real-time Processing**: Instant payment confirmation via webhooks
- **📱 Mobile Optimized**: Perfect for mobile and tablet transactions
- **🎨 Modern UI**: Clean, professional payment interface
- **🌍 New Zealand Built**: Designed specifically for NZ businesses

## 🚀 Quick Start

1. **Install Plugin**: Upload and activate in WordPress
2. **Get API Keys**: Sign up at [tapt.co.nz](https://tapt.co.nz) 
3. **Configure Settings**: Add your API credentials in WooCommerce settings
4. **Set Up Webhooks**: Configure real-time payment notifications
5. **Go Live**: Start accepting modern payments!

## 🛠️ Installation

### Automatic Installation
1. Go to **Plugins > Add New** in WordPress admin
2. Search for "Tapt Payment Gateway"
3. Click **Install** and **Activate**

### Manual Installation
1. Download the plugin ZIP file
2. Upload via **Plugins > Add New > Upload Plugin**
3. Activate the plugin

See [INSTALLATION.md](INSTALLATION.md) for detailed setup instructions.

## ⚙️ Configuration

1. **WooCommerce Settings**
   - Navigate to **WooCommerce > Settings > Payments**
   - Click **Manage** next to "Tapt Payment Gateway"

2. **API Configuration**
   - Get your API keys from [Tapt Dashboard](https://tapt.co.nz/dashboard)
   - Enter API endpoint and key in plugin settings
   - Configure webhook URL for real-time updates

3. **Test Mode**
   - Enable test mode for development
   - Use sandbox credentials for testing
   - Switch to live mode when ready

## 🔄 How It Works

1. **Customer Checkout**: Selects Tapt Payment at checkout
2. **Secure Redirect**: Redirected to Tapt's secure payment page
3. **Payment Options**: Choose NFC tap, QR code, or card payment
4. **Instant Processing**: Payment processed with bank-level security
5. **Real-time Updates**: Order status updated automatically via webhooks
6. **Confirmation**: Customer and merchant receive instant confirmation
