== Description ==

Tapt Payment Gateway brings modern payment processing to your WooCommerce store with support for:

* **NFC Tap-to-Pay**: Customers can tap their phones to pay instantly
* **QR Code Payments**: Quick scanning for contactless transactions  
* **Fixed Fee Structure**: Just $0.25 per transaction, no percentages
* **Bank-Level Security**: PCI compliant with encrypted data transmission
* **Real-time Processing**: Instant payment confirmation via webhooks
* **Mobile Optimized**: Perfect for mobile and tablet transactions

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tapt-payment-gateway` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Settings > Payments
4. Click on "Tapt Payment Gateway" to configure
5. Enter your API credentials from your Tapt dashboard
6. Configure webhook URL in your Tapt admin panel
7. Test the integration and go live!

== Configuration ==

1. **Get API Credentials**: Sign up at https://tapt.co.nz and get your API keys from the admin dashboard
2. **Configure Plugin**: Enter your API endpoint and keys in the WooCommerce payment settings
3. **Set Up Webhooks**: Copy the webhook URL and add it to your Tapt dashboard for real-time updates
4. **Test Payments**: Use test mode to verify everything works correctly

== Frequently Asked Questions ==

= What is Tapt? =

Tapt is a modern payment processing platform that specializes in NFC and QR code payments with transparent, fixed-fee pricing.

= How much does it cost? =

Tapt charges a fixed fee of $0.25 per transaction. No monthly fees, no percentage charges, no hidden costs.

= Do I need special hardware? =

No! Tapt works with any smartphone or tablet that supports NFC. Customers can tap their phones to pay or scan QR codes.

= Is it secure? =

Yes, Tapt is PCI DSS compliant with bank-level security. All payment data is encrypted and processed securely.

= Can I test before going live? =

Absolutely! The plugin includes a test mode so you can verify everything works before processing real payments.

= What payment methods are supported? =

Tapt supports NFC tap-to-pay, QR code payments, and traditional card payments through a secure web interface.

= How do refunds work? =

Refunds can be processed directly from your Tapt dashboard. The plugin will mark refunds appropriately in WooCommerce.
