## 🎯 Quick Start After Download

### 1. Copy LandingPage.tsx
```bash
# Manually copy /components/LandingPage.tsx 
# to /landing-page-package/components/LandingPage.tsx
```

### 2. Install Dependencies
```bash
npm install lucide-react motion react
```

### 3. Add Font
Add to your HTML:
```html
<link href="https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

### 4. Replace Images
Find and replace these 9 imports in `LandingPage.tsx`:
- `logo` - Your logo
- `activeTransactionsImage` - Dashboard phone mockup  
- `paymentStonesImage` - Payment stones image
- `customerPhoneImage` - Customer payment mockup
- `terminalReadyImage` - Quick Payment Entry
- `terminalSplitBillImage` - Split Bill
- `terminalShareImage` - Share Payment
- `terminalNfcImage` - NFC Paywave
- `dashboardImage` - Dashboard screenshot (if used)

### 5. Handle ImageWithFallback
Either:
- Create simple `ImageWithFallback` component (see INSTALLATION.md)
- Or replace all `<ImageWithFallback>` with `<img>` tags

### 6. Use It!
```tsx
import { LandingPage } from './components/LandingPage';

function App() {
  return <LandingPage onGetStarted={() => console.log('Started!')} />;
}
```

## 🆘 Troubleshooting

**Can't find LandingPage.tsx?**
- It's in `/components/LandingPage.tsx` in your current project
- Copy it to `/landing-page-package/components/LandingPage.tsx`

**Images not loading?**
- Check INSTALLATION.md Section 4 for image replacement guide
- Ensure images are in correct path

**Animations not working?**
- Verify `motion` is installed: `npm list motion`
- Check import: `import { motion } from 'motion/react'`

**Tailwind classes not applying?**
- Ensure Tailwind is configured (see INSTALLATION.md)
- Check content paths in tailwind.config.js
