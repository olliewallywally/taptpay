### Component Files

#### `components/LandingPage.tsx` (MAIN COMPONENT)
**Size:** ~1300 lines
**Dependencies:** 
- `lucide-react` (icons)
- `motion/react` (animations)
- `react` (useState, useEffect)
- `HowItWorksSection` component
- 9 image imports (needs to be replaced)

**Sections included:**
1. Hero Section - Logo, tagline, CTA button
2. Everything You Need - Features grid
3. How It Works - Imported component
4. Customer Experience - Payment flow
5. Digital Terminal Features - Carousel
6. Pricing Section - SME & Enterprise tiers
7. Final CTA - Call to action
8. Footer

**Props:**
```typescript
interface LandingPageProps {
  onGetStarted: () => void;
}
```

#### `components/HowItWorksSection.tsx`
**Size:** ~300 lines
**Dependencies:**
- `motion/react`
- `lucide-react` (UserPlus, Sparkles, Smartphone, Check)

**Features:**
- 4-step process timeline
- Color-coded steps
- Mobile: Vertical timeline with gradient connectors
- Desktop: Horizontal 4-column grid
- Hover effects on desktop

**No props required** - Fully self-contained
