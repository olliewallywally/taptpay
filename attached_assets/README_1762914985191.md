# TaptPay Landing Page - Download Package

This package contains all the files needed for the TaptPay landing page.

## 📁 Files Included

### Components
1. **LandingPage.tsx** - Main landing page component
2. **HowItWorksSection.tsx** - "How It Works" section with 4-step timeline

### Dependencies Required

#### NPM Packages
```bash
npm install lucide-react motion
```

#### Libraries Used
- `lucide-react` - For icons (ArrowRight, CheckCircle, Smartphone, Zap, CreditCard, BarChart3, ChevronLeft, ChevronRight)
- `motion/react` - For animations (motion, AnimatePresence)
- `react` - For hooks (useState, useEffect)

## 🔧 Implementation Notes

### ImageWithFallback Component
The landing page uses a custom `ImageWithFallback` component from `/components/figma/ImageWithFallback`. 

If you don't have this component, replace instances with standard `<img>` tags:
```tsx
// Replace this:
<ImageWithFallback src={image} alt="description" style={...} />

// With this:
<img src={image} alt="description" style={...} />
```

### Props Interface
```tsx
interface LandingPageProps {
  onGetStarted: () => void;
}
```

The component expects an `onGetStarted` callback function that triggers when users click the "get started" or "start now" buttons.

## 📱 Responsive Design

The landing page is fully responsive with:
- **Mobile-first approach**
- Fluid typography using `clamp()`
- Different layouts for mobile (`lg:hidden`) and desktop (`hidden lg:grid`)
- Responsive spacing and padding
- Hover effects optimized for desktop

## 🎭 Animations

All sections use Motion (Framer Motion) for:
- Scroll-triggered animations (`whileInView`)
- Entrance animations
- Hover effects
- Carousel transitions

### Key Animation Patterns:
```tsx
initial={{ opacity: 0, y: 50 }}
whileInView={{ opacity: 1, y: 0 }}
viewport={{ once: true }}
transition={{ duration: 0.6 }}
```

## 🚀 Usage

```tsx
import { LandingPage } from './components/LandingPage';

function App() {
  const handleGetStarted = () => {
    // Navigate to login or signup
    console.log('Get started clicked');
  };

  return <LandingPage onGetStarted={handleGetStarted} />;
}
```

## ⚙️ Tailwind Configuration

The landing page uses **Tailwind CSS v4.0** with:
- Custom utilities
- Responsive breakpoints
- No custom `tailwind.config.js` required
- Global styles in `/styles/globals.css`

### Key Tailwind Classes Used:
- Layout: `flex`, `grid`, `hidden lg:grid`, `lg:hidden`
- Spacing: `gap-*`, `px-[5vw]`, `py-[10vh]`
- Colors: `bg-[#0055FF]`, `text-[#00E5CC]`
- Effects: `hover:scale-105`, `transition-all`
